Happy 2015!!!

2015-01-19

Happy New Year! I'm excited to relaunch my personal website. Here's the "stack" I decided to go with:

  • Node + Bower + Gulp
  • GitHub (repository + hosting)
  • Cloudflare (CDN + SSL)
  • Travis CI (Automated deployment)

It's a pretty straightforward stack as the site is static HTML + CSS. I want to thank Sean Farrell for putting together an awesome gulpfile as a base. His whole repo is located here.

Also, thanks BlackTie for creating this awesome bootstrap template that I could modify to fit my needs.
